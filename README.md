# 地图数据可视化系统

一个基于 Web 的地图数据可视化应用，支持在地图上展示数据点、热力图显示、地区筛选和数据导入导出功能。

## 功能特性

1. **地图展示**
   - 自动加载高德地图
   - 标记点模式：在地图上显示数据点标记
   - 热力图模式：根据数据密度生成热力图
   - 标记点聚合：数据量大时自动聚合显示

2. **数据筛选**
   - 按所属区县筛选
   - 按告警名称筛选
   - 按时间范围筛选
   - 实时更新地图显示

3. **数据导入导出**
   - 支持 Excel 文件导入 (.xlsx, .xls)
   - 支持数据导出为 Excel
   - 自动解析经纬度字段

4. **交互功能**
   - 点击标记点查看详细信息
   - 热力图参数调节（半径、透明度）
   - 地图缩放、平移

## 数据格式

导入的 Excel 文件需要包含以下字段：

| 字段名 | 说明 | 示例 |
|--------|------|------|
| 基站号 | 基站唯一标识 | 11555342 |
| BBU名称 | 设备名称 | H11555342-甬台温福高铁... |
| 告警名称 | 告警类型 | GNSS星卡锁星不足告警 |

| 所属站点 | 站点名称 | 温州瓯海潘桥新陈庄 |
| 所属区县 | 行政区 | 瓯海区 |
| 经度 | 地理经度 | 120.574411 |

## 使用说明

### 1. 申请高德地图 API Key

本项目使用高德地图 JavaScript API，需要申请 Key：

1. 访问 [高德地图开放平台](https://lbs.amap.com/)
2. 注册账号并登录
3. 进入「控制台」→「应用管理」→「创建应用」
4. 添加 Key，选择「Web端 (JS API)」
5. 复制 Key 替换 `index.html` 中的 `您的高德地图Key`

### 2. 启动项目

#### 方式一：使用 Python 启动本地服务器

```bash
# Python 3
python -m http.server 8080

# Python 2
python -m SimpleHTTPServer 8080
```

然后访问 http://localhost:8080

#### 方式二：使用 VS Code Live Server 插件

安装 Live Server 插件，右键点击 `index.html` 选择「Open with Live Server」

### 3. 导入数据

1. 点击「导入数据」按钮
2. 选择 Excel 文件（.xlsx 或 .xls）
3. 系统自动解析并在地图上显示

### 4. 使用筛选

1. 在左侧筛选面板选择条件
2. 点击「应用筛选」按钮
3. 地图自动更新显示筛选结果

### 5. 切换地图模式

- 标记点模式：显示具体的数据点位置
- 热力图模式：显示数据密度分布

### 6. 导出数据

点击「导出数据」按钮，将当前筛选后的数据导出为 Excel 文件。

## 技术栈

- **前端**: HTML5, CSS3, JavaScript (ES6+)
- **地图服务**: 高德地图 JavaScript API 2.0
- **Excel处理**: SheetJS (xlsx.js)
- **UI组件**: 原生 CSS

## 浏览器兼容性

- Chrome 60+
- Firefox 60+
- Safari 12+
- Edge 79+

## 注意事项

1. 高德地图 API Key 有日调用量限制，免费版一般足够个人使用
2. 导入的 Excel 文件必须包含「经度」和「纬度」字段
3. 经纬度坐标系应为 WGS-84 或 GCJ-02（火星坐标系）
4. 建议数据量不超过 10000 条，以保证性能
