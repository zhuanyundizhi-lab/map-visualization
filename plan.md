# 地图数据可视化网页项目计划

## 项目概述
设计一个网页前端，在地图中展示导入的数据，支持热力图、地区筛选、数据导入导出等功能。

## 数据格式分析
参考文件：`3月15日后锁星不足告警清单.xlsx`

### 数据字段
| 字段名 | 类型 | 说明 |
|--------|------|------|
| 基站号 | int64 | 基站唯一标识 |
| BBU名称 | object | 设备名称 |
| 告警名称 | object | 告警类型 |
| 发生时间 | object | 告警发生时间 |
| 机房名称 | object | 机房位置 |
| 所属站点 | object | 站点名称 |
| 所属区县 | object | 所属行政区 |
| 经度 | float64 | 地理经度 |
| 纬度 | float64 | 地理纬度 |

## 功能需求

### 1. 地图加载
- 自动加载在线地图（使用高德地图或百度地图 API）
- 支持地图缩放、平移等基本操作
- 地图中心点根据数据自动调整

### 2. 数据展示
- 将每个数据点的经纬度映射到地图上
- 使用标记点展示位置
- 点击标记显示详细信息（地址、时间等）
- 支持标记点聚合（当数据量大时）

### 3. 热力图功能
- 支持热力图模式切换
- 根据数据密度生成热力图
- 可调节热力图参数（半径、透明度等）

### 4. 地区筛选
- 按所属区县筛选数据
- 按时间范围筛选
- 按告警名称筛选
- 筛选结果实时更新地图显示

### 5. 导入导出功能
- 支持 Excel 文件导入（.xlsx 格式）
- 支持数据导出为 Excel
- 导入时自动解析经纬度字段

## 技术栈
- **前端框架**: HTML5 + CSS3 + JavaScript (原生)
- **地图服务**: 高德地图 JavaScript API
- **UI组件**: 原生 CSS + 自定义组件
- **数据处理**: SheetJS (xlsx.js) 用于 Excel 解析
- **热力图**: 高德地图热力图插件

## 实现步骤

### Phase 1: 基础架构
1. 创建 HTML 页面结构
2. 引入高德地图 API
3. 初始化地图
4. 添加基础样式

### Phase 2: 数据展示
1. 实现标记点显示
2. 添加信息窗口
3. 实现标记点聚合

### Phase 3: 热力图
1. 集成热力图插件
2. 实现热力图切换
3. 添加热力图参数控制

### Phase 4: 筛选功能
1. 实现区县筛选
2. 实现时间筛选
3. 实现告警类型筛选

### Phase 5: 导入导出
1. 实现 Excel 导入
2. 实现数据导出
3. 添加文件上传组件
